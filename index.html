Session

/clearBetween unrelated tasks. Clean reset > lossy compact.
/btwSide questions without growing context. Zero cost.
/renameName sessions like branches. Resume later with --resume.
Esc EscRewind to any checkpoint. Restore code, conversation, or both.
/effortmedium for routine. high for deep work. Toggle freely.

Workflow

1State work type upfront. "Quick question" / "feature spec" / "code session" / "QA run".
2Interview before building. "Interview me using AskUserQuestion. Dig into edge cases." Spec in a fresh session after.
3Writer/Reviewer split. One session writes. A fresh one reviews. No self-bias.
4Give verification. Tests, screenshots, expected output. Single highest-leverage habit.
5Explore > Plan > Implement > Commit. Plan Mode for unknowns. Skip for small fixes.
6Two corrections max. Third fail = /clear, rewrite the prompt with what you learned.

Tokens

$Max 2 parallel agents. Each loads ~7K overhead. Bypass only with high budget.
$CLI > MCP. gh curl jq — no schema bloat.
$Disable unused plugins. Each adds 300-500 tokens of tool schemas.
$Subagents for research. Separate context, summary back. Main stays clean.
$Scope narrowly. "Check src/auth/" not "investigate the codebase".
$No redundant reads. Already read this session? Skip unless edited.

Setup

*CLAUDE.md: less is more. "Would removing this cause mistakes?" No? Cut it.
*Hooks for musts. CLAUDE.md ~80% adherence. Hooks = 100%. Safety goes in hooks.
*Memory cap: 15 files. Merge or delete before adding. MEMORY.md is an index only.
*Skills are folders. Include scripts, data, assets. Claude discovers progressively.
*Gotchas section = gold. Build from failure points. Update after every recurring mistake.
*Push CLAUDE.md to git. It's shared context. Compounds over time across your team.

Don't

XKitchen sink sessions. One task drifts into three. Context fills with noise. /clear between.
XCorrection loops. 3+ corrections = polluted context. Rewrite the prompt, don't keep fixing.
XBloated CLAUDE.md. Over 5KB and rules get ignored. Prune. Move domain knowledge to skills.
XUnscoped exploration. "Investigate everything" = hundreds of files read. Scope or use subagents.
XShip without verification. Plausible code misses edge cases. No tests = no confidence.
